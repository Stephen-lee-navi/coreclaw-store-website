# CoreClaw Store

CoreClaw Store - 网络爬虫和自动化工具市场

## 简介

CoreClaw 是一个提供网络爬虫和自动化工具的平台，帮助用户轻松获取各种网站数据。

## 功能特点

- 按平台浏览 Workers（Google, Amazon, YouTube, TikTok, Facebook 等）
- 按业务场景浏览 Workers（电子商务, 社交媒体, 招聘求职, 地图服务等）
- 发布 Workers 并获得持续收益
- 完整的数据抓取技术栈

## 技术栈

- HTML5
- CSS3
- JavaScript

## 访问地址

部署后可通过以下地址访问：
https://coreclaw-store.vercel.app

## 本地运行

```bash
# 克隆项目
git clone https://github.com/Stephen-lee-navi/coreclaw-store-website.git

# 进入项目目录
cd coreclaw-store-website

# 启动本地服务器
python -m http.server 8080

# 访问 http://localhost:8080
```
