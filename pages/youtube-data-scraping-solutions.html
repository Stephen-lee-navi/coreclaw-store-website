常见问题

YouTubeScraper 可以抓取哪些数据？ ▼
YouTube Scraper 通常可以采集公开视频信息、频道资料、搜索结果、评论、字幕、播放列表、Shorts 数据以及部分互动指标。具体支持哪些字段，取决于工具能力和目标内容是否公开可访问。
YouTube Scraper 最常见的用途是什么？ ▼
最常见的用途包括视频趋势研究、竞品频道监控、评论分析、字幕提取、SEO 选题、达人评估以及 Shorts 内容追踪。
YouTube Scraper 和 YouTube API 有什么区别 ▼
YouTube Scraper 更适合采集公开页面数据，用于内容研究、竞品分析和趋势追踪；YouTube API 则更适合获取官方授权范围内的数据，通常需要开发者配置和配额管理。
不会写代码也能使用 YouTube Scraper 吗？ ▼
可以。Coreclaw提供的抓取工具可以直接输入视频链接、频道链接或关键词开始采集，而不是自己开发程序。
使用 YouTube Scraper 需要登录 YouTube 吗？ ▼
这取决于具体工具和目标数据类型。对于公开内容，通常不一定需要登录；但某些功能或访问场景，可能需要额外配置或认证信息。
