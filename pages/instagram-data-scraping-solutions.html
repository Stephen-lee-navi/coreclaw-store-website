# 示例代码：使用 CoreClaw API 抓取 Instagram 帖子数据
import requests

api_key = "your_api_key_here"
url = "https://api.coreclaw.com/v1/instagram/post/scrape"

payload = {
    "post_url": "https://instagram.com/p/example123",
    "include_comments": True,
    "include_likes": True
}

headers = {
    "Authorization": f"Bearer {api_key}",
    "Content-Type": "application/json"
}

response = requests.post(url, json=payload, headers=headers)
data = response.json()

print(data)

常见问题

Instagram Scraper 通常可以采集公开可见的账号主页信息、帖子内容、评论、标签内容、Reels 数据，以及部分互动指标。具体支持哪些字段，取决于工具能力和目标页面是否公开可访问。

取决于具体工具。有些工具可以直接抓取公开数据，有些功能可能需要登录状态或额外配置。对于大多数用户来说，是否必须登录会直接影响上手成本。
Instagram Scraper 最常见的用途是什么？ ▼
最常见的用途包括达人筛选、竞品监控、爆款内容研究、评论情绪分析、标签趋势追踪，以及 Reels 内容表现分析。

可以。账号主页数据通常是用户最先需要的内容之一，比如简介、粉丝数、发帖量、外链、认证状态和账号类型，这些数据常用于达人评估和竞品研究。

合规性主要取决于数据是否公开、是否涉及个人信息，以及您的使用方式是否符合法律法规和平台规则。对于大多数用户来说，确保数据来源透明、用途合理，是使用 Instagram Scraper 时最重要的前提之一。
