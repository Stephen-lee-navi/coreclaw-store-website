谷歌数据抓取演示

专为开发者开发，输出结构化、可分析的数据，方便快速接入你的工作流

# 示例代码：使用 CoreClaw API 抓取 Google Maps 数据
import requests

api_key = "your_api_key_here"
url = "https://api.coreclaw.com/v1/google-maps/scrape"

payload = {
    "keyword": "restaurant",
    "location": "New York, NY",
    "limit": 100
}

headers = {
    "Authorization": f"Bearer {api_key}",
    "Content-Type": "application/json"
}

response = requests.post(url, json=payload, headers=headers)
data = response.json()

print(data)
