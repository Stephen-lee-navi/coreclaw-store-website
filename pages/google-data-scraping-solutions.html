谷歌数据抓取演示

专为开发者开发，输出结构化、可分析的数据，方便快速接入你的工作流

# 示例代码：使用 CoreClaw API 抓取 Google Maps 数据
import requests

api_key = "your_api_key_here"
url = "https://api.coreclaw.com/v1/google-maps/scrape"

payload = {
    "keyword": "restaurant",
    "location": "New York, NY",
    "limit": 100
}

headers = {
    "Authorization": f"Bearer {api_key}",
    "Content-Type": "application/json"
}

response = requests.post(url, json=payload, headers=headers)
data = response.json()

print(data)

什么是Google数据？ ▼
Google平台数据通常指来自Google生态的数据，例如Google Maps、Google Player Store、Google Shopping等。

用户可以获取的数据包括：
- 商家信息（名称、地址、电话）
- 用户评价与评分
- 搜索排名与关键词数据

如何抓取Google Places API数据? ▼
如果使用官方 API，一般需要先开通计费并启用 Places API，再通过接口获取地点与详情数据；但官方API价格昂贵，如果你更关注批量采集、结构化导出和自动化处理，Google Scraper 会是更灵活、更具性价比的选择。
Google数据抓取是否合法？ ▼
通常情况下：
- 抓取公开数据是允许的
